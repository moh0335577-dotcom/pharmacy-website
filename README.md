# 💊 صيدلية شهد عبد المنعم إسماعيل

موقع احترافي لصيدلية حديثة مع تصميم UI/UX متقدم

## 🎯 الميزات

- ✨ تصميم حديث واحترافي
- 🎨 ألوان طبية هادئة
- 📱 متوافق مع جميع الأجهزة (Responsive)
- ⚡ أداء عالي جداً
- 🎭 حركات وأنيميشن سلسة
- 🔧 كود نظيف وقابل للصيانة

## 🚀 الخطوات الأولى

### المتطلبات
- Node.js 16+ 
- npm أو yarn

### التثبيت

```bash
# استنساخ المشروع
git clone https://github.com/moh0335577-dotcom/pharmacy-website.git

# الدخول للمجلد
cd pharmacy-website

# تثبيت المكتبات
npm install

# تشغيل خادم التطوير
npm run dev
```

### البناء للإنتاج

```bash
npm run build
```

## 📁 هيكل المشروع

```
src/
├── components/        # مكونات React
│   ├── Button/       # زر قابل للتخصيص
│   ├── Card/         # بطاقة عامة
│   ├── Container/    # حاوية
│   ├── Navbar/       # القائمة العلوية
│   ├── Hero/         # القسم الرئيسي
│   ├── Services/     # قسم الخدمات
│   └── Footer/       # تذييل الموقع
├── constants/        # ثوابت المشروع
├── utils/           # دوال مساعدة
├── styles/          # أنماط عامة
├── App.jsx          # المكون الرئيسي
└── main.jsx         # نقطة الدخول
```
